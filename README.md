# 记账助手 (鸿蒙ArkTs版)

使用鸿蒙 ArkTs Stage 模型开发实现的记账助手App，数据使用鸿蒙本地关系型数据库进行独立存储，不依赖服务。
已实现 "当月记账数据"，"添加记账"，"数据统计" 等功能.
